Web 1.0

1990 En HTML y la web 1.0 Tim Berners-Lee define por fin el HTML, crea el primer navegador web, Viola WWW, que funcionara en modo texto, y el uso de fromesets o Marcos ademas de efectos como el parpadeo y las marquesinas, botones Gif, etc. Hacian mas dinamica a la red

1991 Surgimiento de la Web 1.0 Consiste en un bonche de documentos navegadores visuales como IE, web web conectados por medio de hipervinculos, el usuario no podia interactuar.

1993 La web y su uso gratuito para todo el publico. La web 1.o era estatica y por lo general unicamente informativa. Los usuarios eran solo lectores. Interseccion reducida a formularios de contacto, inscripciones y boletines. Sitios direccionales y no colaborativos.
